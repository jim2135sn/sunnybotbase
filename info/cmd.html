SunnyBot Commands

Audio

/clear Clears all songs in queue.
/connect Makes the bot join a voice channel.
/debug A user command that gets debug information about the bot.
/disconnect Makes the bot leave a voice channel.
/list List all songs in queue.
/pause Pause the current audio playing.
/play [song name/URL] Plays a music url in a voice channel.
/playing Sees the song that is in the queue.
/remove Remove a song in the queue.
/seek Seeks to the seconds in the currently playing song.
/stop Stop audio that is currently playing.
/volume [0-100] Sets the volume of the currently playing song.

FunOrRandom

/8ball The magic 8ball command. Used for yes or no answers.
/avatar Get the user's avatar.
/buzzify Generates a buzz lightyear meme.
/d Roles a dice between two values.
/echo Makes the bot say anything, and I mean anything.
/meme Displays a random meme.
/ping Pings the bot, along with latency information.
/ship Ships you and another user!
/to Converts one format to another.
/translate Translates a message based on message id or text.
/tts Says something through GTTS/text-to-speech (send an attachment w...).

Games

/poke Query Information about the pokemon franchise.
/rank View rank information.
/rankconfig Configurate rank data for this server.

Moderation

/ban Bans a user.
/info Information command for checking servers & Member.
/kick Kicks a user from the server.
/logs Lists audit logs entries for a member.
/purge Purge a certain amount of messages in a channel.
/slowmode Set the channel slowmode.
/voicekick Kicks a user from the voice channel.
